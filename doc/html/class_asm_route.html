Detailed Description

permet de construire un tableau servant à router les requêtes

Équivalant à un front-controller

Constructor & Destructor Documentation

__construct ( )

mise en place de la donnée membre $this->route

Le tableau permet de mapper une URI à un groupe controlleur/méthode/template

  • controller: Nom du controlleur à appeler, le nom de la classe, sans "Controller" à la fin (exemple, StaticPage, User)
  • action: Méthode qui serra appelé dans la classe du controlleur
  • template: nom du template à appeler (sans le .html)

la variable se présente donc sous la forme, avec la page d'acceuil comme exemple :

$this->route = array(
'/' => array(
'controller' => 'StaticPage',
'action' => 'home',
'template' => 'home'
),
// Autres routes...
)
