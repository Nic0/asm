ASM: Agregation de Services Monitoring

# Dependances

- Support de yaml:
    sudo apt-get install libyaml-dev
    sudo pecl install yaml

- Support ldap
    sudo apt-get install php-net-ldap

- Support SNMP
    sudo apt-get install php5-snmp
    sudo apt-get install snmp-mibs-downloader
